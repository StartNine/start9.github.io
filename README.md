## Update SVG Symbols
- Go to [icomoon](https://icomoon.io/app/)
- Import ``selection.json``
- Choose ``Generate SVG & More``
- Replace ``assets/symbol-defs.svg`` with the one you just downloaded
- Replace ``selection.svg`` with the one you just downloaded
